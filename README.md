# Proyecto Java

Este es un proyecto Java básico con la siguiente estructura:

```
.
├── src/        # Código fuente
├── bin/        # Archivos compilados
└── README.md   # Este archivo
```

## Compilación

Para compilar el proyecto, ejecuta:
```bash
javac -d bin src/**/*.java
```

## Ejecución

Para ejecutar el programa, usa:
```bash
java -cp bin Main
```
